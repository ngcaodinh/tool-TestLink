Report Contents

The generated Word document will include the following sections:

  • 1. Table of Contents
  • 2. Report Information (Plan, Project, Build details)
  • 3. Test Suite Descriptions
  • 4. Test Case Summary (name, external ID, version)
  • 5. Test Case Scenarios (steps as numbered list)
  • 6. Test Case Authors
  • 7. Test Case Keywords
  • 8. Test Case Custom Fields
  • 9. Test Case Related Requirements
  • 10. Execution Notes
  • 11. Step Execution Notes
  • 12. Test Results (color-coded: Pass/Fail/Blocked)
  • 13. Step Execution Results
  • 14. Build Custom Fields
  • 15. Metrics Summary (Pass Rate, Fail Rate, etc.)

Note: Reports are generated using data from your TestLink database. Ensure that the TestLink XML-RPC API and MySQL database are accessible for complete data.
